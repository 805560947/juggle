# juggle

## 介绍
    阿里推出飞冰来进行页面搭建，但是飞冰的物料开发不是想象中的简单。
    开发此项目，可以直接使用市面上线程的组件库（iview, vux, Element），也可以基于现有组件库封装自定义组件。
    通过拖拽布局，生成页面配置 json 文件，再通过json动态渲染页面。
